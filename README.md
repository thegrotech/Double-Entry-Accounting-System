# Accounting System Backend

Professional Double-Entry Accounting System with PostgreSQL.

## Features
- Double-entry bookkeeping
- Balance Sheet & Income Statement
- User authentication (Admin/Viewer roles)
- PostgreSQL database
- REST API

## Deployment
- **Backend**: Render.com
- **Database**: Supabase PostgreSQL
- **Frontend**: Vercel

## Setup
1. Clone repository
2. `npm install`
3. Create `.env` file
4. `node database/init-pg.js`
5. `npm start`
